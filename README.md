<p dir="auto">Use python if:</p>
<ol dir="auto">
<li>Your server's CPU has one core or you want to run proxy on one core.</li>
<li>Your server is low-end.</li>
<li>You are serving a small group of people. (Like family or small company)</li>
<li>You want to limit user connections.</li>
<li>You also have another application or service running on your server. (Openvpn, shadowsocks, nginx or ...)</li>
</ol>
<p dir="auto">Otherwise, use official proxy.</p>
